アプリの説明
====================
2020年に開発した卒論用のAndroidアプリで、開発言語は Java です。卒論のテーマは日本語学習者を対象に、日本語テ系動詞の習得を検証することです。
当時アプリ三つ作りましたが、これはその中のゲーム式のアプリで、具体的な説明は以下の通りです。
